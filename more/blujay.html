BLUJAY

INTRO

Client asked for a landing page that stayed consistent design-wise and followed the branding. The template required different sections for different purposes. Also this template would be reused for multiple pages.

PROCESS

Having all the information I needed, I went to work. I started by browsing through the BluJay site. I visited multiple pages, while noticing why and how certain things were designed.

I sent over two high fidelity mockups to the client. They chose one. From there, I built the detailed mockup.

Despite the constraints, I included some personal touches that the client ended up loving, like the three leaf design.

In addition to being the sole designer, I was also the developer for this project. The landing page was built for Adobe Marketo. I utilized HTML, CSS, and Javascript to achieve this landing page template.

This template was designed in Sketch and built with Visual Studio Code
